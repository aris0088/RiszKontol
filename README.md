
# RISZ OFFICIAL PAYMENT GATEWAY

## Ready for Vercel Deploy

### Features
- Next.js 14 App Router
- Tailwind Dark UI
- User system (ready)
- Balance & API Key
- QRIS & Withdraw structure
- Secure & scalable

### Deploy
1. npm install
2. cp .env.example .env
3. npm run build
4. Deploy to Vercel

### Notes
QRIS & Withdraw require official payment provider (Midtrans/Xendit).
